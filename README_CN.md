## 🎯 什么是 MaudelClaw？

**MaudelClaw** 是一个**跨平台 AI Agent 智能体系统**，专为注重隐私的用户设计，在享受强大 AI 能力的同时保护数据安全。MaudelClaw 原生支持 Windows、macOS 和 Web 平台，让智能自动化触手可及。

## ✨ 核心优势

### 🎨 界面美观、交互流畅
- 现代化赛博朋克风格设计
- 流畅的动画和响应式交互
- 深色模式优化，开发者友好
- 可自定义主题和布局

### ⚡ 一键安装、开箱即用
- **零配置** - 下载即用，无需复杂设置
- 无需手动安装依赖
- 自动检测和配置 AI 模型
- 内置自动更新功能

### 🔧 功能完善、能力强大
- **浏览器自动化**：自动登录、网页抓取、截图
- **文档处理**：支持 PDF、Word、PPT、Excel
- **邮件管理**：SMTP 发送 + IMAP 接收
- **语音输入**：本地 Whisper 语音识别
- **图像处理**：生成、分析、编辑
- **网络搜索**：实时信息检索
- **知识库 RAG**：智能文档问答
- **记忆系统**：长期对话上下文
- **SKILL 系统**：创建和分享专家知识

## 🚀 快速开始

### 下载安装

选择你的平台：

| 平台 | 系统要求 | 下载 |
|------|----------|------|
| **macOS** | macOS 10.15+ | [下载 .dmg](./releases/latest) |
| **Windows** | Windows 10+ | [下载 .exe](./releases/latest) |
| **Web** | 任意现代浏览器 | [启动网页版](https://maudelclaw.ai) |

### 系统要求

- **内存**：8GB 最低（推荐 16GB）
- **存储**：2GB 可用空间
- **网络**：云端模型需要网络连接
- **可选**：支持本地 LLM（需安装 Ollama）

## 📦 安装包

所有发布版本均可在 [Releases](./releases) 页面下载：

- **macOS**：`.dmg` 安装包
- **Windows**：`.exe` 安装包
- **便携版**：无需安装，直接运行

## 🔐 隐私与安全

- **本地优先架构**：数据永不离设备
